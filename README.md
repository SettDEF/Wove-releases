# Wove — Releases

Public downloads for **[Wove](https://github.com/SettDEF/Wove)** — a fast, local-first music player for Android.

## Install
1. Download the latest **`Wove-*.apk`** from the [Releases](https://github.com/SettDEF/Wove-releases/releases) page.
2. Open it on your phone and allow "install unknown apps" if prompted.

**Auto-updates:** add this repo to [Obtainium](https://github.com/ImranR98/Obtainium) and it'll keep Wove up to date.

`latest.json` here is the manifest the in-app updater reads.
